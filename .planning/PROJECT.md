# Amrit Palace Clone

## What This Is
A full web application clone of amritpalace.com, capturing its exact layout, aesthetic, and styling. The project will replicate the high-end fine dining aesthetic with a high-contrast dark and light theme, staggered/alternating block layout, sticky header, full viewport hero section, and menu item layouts. Images and copy will act as placeholders to be replaced later by the user.

## Core Value
Provides a visually stunning, premium template mimicking amritpalace.com that allows the user to simply swap out text and images to have a production-ready luxury restaurant website.
