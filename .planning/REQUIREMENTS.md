# Requirements

## 1. Goal
Provide an exact visual clone of the amritpalace.com website layout and styling to be used as a custom base template, ready for image and menu text replacements.

## 2. Scope

| Feature | In Scope | Out of Scope |
|---------|----------|-------------|
| **Theme** | Exact color mapping (Cream, Charcoal, Burnt Orange), typography pairs, visual aesthetics. | Original business fonts if licensed strictly (will use close Google Web Font equivalents like Cormorant Garamond & Montserrat). |
| **Responsive** | Desktop, tablet, mobile layouts. | |
| **Header** | Sticky header with scroll-based background transition. | Functional login or ordering. |
| **Hero** | Full page banner image handling with text overlays. | Slider functionality if not present in reference. |
| **Menu** | Sticky category sub-nav, styled list items. | Database-driven menu data (using static HTML placeholders). |
| **Footer** | 4-column dark footer. | Newsletter subscribe functional backend. |

## 3. Tech Stack
- HTML5 (Vite/React or pure HTML based on the environment found).
- Vanilla CSS (for layout and styling matches).
- No complex component libraries to ensure exact styling.
